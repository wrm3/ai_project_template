# Grant Application Timeline Template

## SBIR Phase I Timeline

### Month 1-2: Preparation Phase

**Week 1-2: Research and Planning**
- [ ] Identify relevant agency and solicitation topic
- [ ] Read solicitation thoroughly (requirements, deadlines)
- [ ] Search past awarded projects (NIH RePORTER, SBIR.gov)
- [ ] Contact program officer (ask about fit, get feedback)
- [ ] Assess team capability and gaps
- [ ] Estimate budget and resources needed

**Week 3-4: Feasibility Assessment**
- [ ] Confirm technical feasibility
- [ ] Verify eligibility requirements (US ownership, <500 employees)
- [ ] Gather preliminary data (if available)
- [ ] Identify consultants/subcontractors needed
- [ ] Confirm availability of key personnel
- [ ] Decide go/no-go on application

### Month 3-4: Proposal Development

**Week 5-6: Technical Approach**
- [ ] Draft technical objectives (specific, measurable)
- [ ] Outline methods and approach
- [ ] Create timeline/Gantt chart
- [ ] Identify potential pitfalls and alternatives
- [ ] Develop preliminary data section
- [ ] Create figures and diagrams

**Week 7-8: Commercialization Plan**
- [ ] Market analysis and sizing
- [ ] Competitive landscape assessment
- [ ] Business model and revenue projections
- [ ] Go-to-market strategy
- [ ] Phase II and Phase III plans
- [ ] Team and funding strategy

### Month 5: Budget and Supporting Documents

**Week 9: Budget Development**
- [ ] Personnel costs (PI, staff, fringe benefits)
- [ ] Equipment and supplies
- [ ] Consultant/subaward budgets
- [ ] Travel (conferences, customer discovery)
- [ ] Other direct costs
- [ ] Indirect costs (F&A rate)
- [ ] Budget justification (detailed)

**Week 10: Supporting Documents**
- [ ] Biosketches (NIH format for all key personnel)
- [ ] Letters of support (customers, partners)
- [ ] Company information (incorporation date, employees)
- [ ] Facilities and equipment description
- [ ] Prior SBIR/STTR awards (if any)

### Month 6: Review and Submission

**Week 11: Internal Review**
- [ ] Complete first full draft
- [ ] Internal team review
- [ ] Revise based on feedback
- [ ] Check against solicitation requirements (compliance)
- [ ] Proofread for typos and clarity

**Week 12: External Review and Finalization**
- [ ] External reviewer feedback (advisor, consultant)
- [ ] Final revisions
- [ ] Format check (fonts, margins, page limits)
- [ ] Prepare all required forms
- [ ] Get institutional signatures/approvals
- [ ] Upload to grants portal (grants.gov or agency-specific)
- [ ] **Submit by deadline** (usually 5pm ET)

### Month 7-12: Review Period

**Month 7-9: Initial Review**
- [ ] Agency assigns reviewers (3-5 typically)
- [ ] Reviewers score proposal
- [ ] Preliminary scores released (some agencies)

**Month 9-11: Council/Advisory Review**
- [ ] Advisory council review (NIH)
- [ ] Final scoring and ranking
- [ ] Funding line determined

**Month 12: Award Notification**
- [ ] Award or decline notification
- [ ] If awarded: Accept award, set up account
- [ ] If declined: Request reviewer comments
- [ ] Decide whether to resubmit (address comments)

### Post-Award (If Funded)

**Month 13-24: Phase I Execution**
- [ ] Hire personnel
- [ ] Order equipment and supplies
- [ ] Execute technical plan
- [ ] Monthly progress reports
- [ ] Engage with program officer
- [ ] Customer discovery and validation
- [ ] Prepare for Phase II application

**Month 18: Prepare Phase II Application**
- [ ] Review Phase I progress
- [ ] Develop Phase II objectives (24-month plan)
- [ ] Update commercialization plan
- [ ] Prepare Phase II budget ($750k-$1.5M)
- [ ] Submit Phase II application

**Month 24: Phase I Completion**
- [ ] Final report submission
- [ ] Phase II award notification (if applied)

## Key Dates by Agency (Examples)

### NIH
- **Deadlines**: April 5, September 5, January 5 (standard)
- **Review Period**: 6-9 months
- **Total Time to Award**: 9-12 months

### NSF
- **Deadlines**: Rolling or twice per year (check website)
- **Review Period**: 4-6 months
- **Total Time to Award**: 6-9 months

### DOD
- **Deadlines**: Varies by solicitation (check SBIR.gov)
- **Review Period**: 4-8 months
- **Total Time to Award**: 6-12 months

### DOE
- **Deadlines**: Annual solicitations (usually Q2)
- **Review Period**: 6-9 months
- **Total Time to Award**: 9-12 months

## Effort Estimation

- Research and planning: 20 hours
- Technical approach: 50 hours
- Commercialization plan: 40 hours
- Budget and supporting docs: 20 hours
- Review and revision: 20 hours

**Phase II Application** (total ~200 hours):
- More extensive than Phase I
- Build on Phase I results
- Expanded commercialization plan

- Phase I: $5,000-$15,000
- Phase II: $10,000-$25,000

## Common Timeline Mistakes

**Don't**:
- Start 2 weeks before deadline (need 3+ months)
- Forget program officer contact (do early)
- Skip preliminary data (weakens proposal)
- Ignore reviewer comments (if resubmitting)
- Apply to wrong agency/topic (poor fit)
- Submit on deadline day (technical issues common)

**Do**:
- Start 3-4 months early
- Contact program officer in month 1
- Collect preliminary data throughout
- Apply to multiple agencies (increase odds)
- Submit 2-3 days early (avoid last-minute issues)
- Get external review before submission
