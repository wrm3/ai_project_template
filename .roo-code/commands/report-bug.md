# Roo-Code Command: report-bug

Report and track a bug in the system.

## Usage
```
/report-bug Brief bug description
```

## Process
1. Guided bug report questionnaire
2. Bug file creation in `.fstrent_spec_tasks/BUGS.md`
3. Task creation for fix (if needed)
4. Severity classification
5. Assignment suggestions

## Example
```
/report-bug Login fails with special characters in password
```

## Information Gathered
- Steps to reproduce
- Expected vs actual behavior
- Environment details
- Error messages/logs
- Severity level
- Affected users

## Outputs
- BUGS.md entry
- Optional task file for fix
- Investigation notes
