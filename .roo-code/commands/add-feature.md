# Roo-Code Command: add-feature

Add a new feature to the project with full planning.

## Usage
```
/add-feature Feature description
```

## Process
1. Requirements gathering
2. PRD creation
3. Task breakdown
4. Feature file generation
5. Dependencies identification

## Example
```
/add-feature Real-time notifications with WebSocket
```

## Creates
- Feature document in `.fstrent_spec_tasks/features/`
- Multiple task files for implementation
- Updates PLAN.md and TASKS.md
- Documents dependencies and acceptance criteria
