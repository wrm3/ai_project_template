# Roo-Code Command: start-planning

Begin feature planning with structured questionnaire.

## Usage
```
/start-planning Feature name
```

## Process
1. Guided questionnaire for requirements
2. PRD generation in `.fstrent_spec_tasks/PLAN.md`
3. Scope validation
4. Task breakdown suggestions
5. Feature file creation in `.fstrent_spec_tasks/features/`

## Example
```
/start-planning User authentication system
```

## Outputs
- Updated PLAN.md with feature section
- Feature file: `.fstrent_spec_tasks/features/user-authentication.md`
- Suggested task breakdown
- Acceptance criteria template
